## Planner Mobile

Aplicação mobile do **Planner**, um app para **planejamento e organização de viagens**.

O app permite que usuários criem viagens, organizem atividades por dia e compartilhem informações importantes com outras pessoas envolvidas.

## 🎥 Demonstração

![Planner App Demo](./docs/app-v-1.0.0-gif.gif)

## 🎯 Objetivo do app

O Planner nasceu para resolver um problema comum ao planejar viagens:

- Informações espalhadas em conversas
- Links importantes perdidos
- Falta de organização das atividades por dia
- Dificuldade de compartilhar tudo com quem vai viajar junto

Com o app, a ideia é centralizar tudo em um único lugar, de forma simples e acessível pelo celular.

## ✨ Funcionalidades atuais

- Autenticação de usuário
- Cadastro de novos usuários
- Listagem de viagens
- Detalhamento da viagem:
  - Atividades organizadas por dia
  - Links úteis relacionados à viagem
  - Lista de convidados

Mesmo em sua versão inicial, o app já é funcional e pode ser usado para planejar uma viagem real.

## 🚀 Tecnologias

- React Native

- TypeScript
- NativeWind

## ⚙️ Requisitos

- Node.js >= 20
- Expo CLI
- App Expo Go (Android ou iOS)

## ▶️ Como rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/RodrigoGrz/planner-mobile
cd planner-mobile
```

### 2. Rodando app

```bash
npm run start
```

### 3. Rodando testes

```bash
npm run test
```
